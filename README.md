# Fraternité Crêpe Catering Website

## Overview
This project is a complete institutional and catering service website developed for **Fraternité Crêpe**, focused on presenting catering services, interactive menus, and customer quote requests in a modern and responsive way.

The platform was built to provide an elegant user experience, showcasing the company's menu options while allowing users to request personalized event quotations directly through the website.

---

## Features
- Fully responsive design optimized for desktop, tablet, and mobile devices
- Interactive catering menu modal with animated transitions
- Dynamic menu navigation with keyboard and outside-click support
- Smooth UI animations and responsive layouts
- Quote request form integrated with PHP and MySQL
- Secure login system using prepared statements
- Environment variable configuration using `.env`
- Organized and scalable project structure

---

## Project Structure
- **css/** → Global styles, responsive layouts, and animations
- **js/** → Interactive behaviors, menu controls, and modal logic
- **pages/cardapios/** → Catering menu pages and modal content
- **php/** → Backend logic, authentication, and database connection
- **vendor/** → Composer dependencies
- **tests/** → Testing and experimental layouts

---

## Getting Started

### Clone the repository

git clone https://github.com/Bortolans/fratSite

## Live Demo

[(link do netlify)](https://fraternite-crepe-site.netlify.app/)

### Install PHP dependencies

composer install

### Configure environment variables

Create a `.env` file in the project root

### Run the project

Use XAMPP, Laragon, or another local PHP server environment.
